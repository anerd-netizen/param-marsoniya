## 📦 How to Setup My Projects
For security purposes, I do not store sensitive API keys in my public repositories. To run my projects locally:

1. Clone this repository.
2. Create a `config.js` file in the root directory.
3. Add your own API key to the file: `const apiKey = 'YOUR_KEY_HERE';`
4. Ensure `config.js` is added to your `.gitignore` file to keep your credentials safe.
